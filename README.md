# Под Огромной Луной

Рабочий проект романа.

Здесь хранятся:
- главы романа
- краткое содержание книги
- карта персонажей
- темы, мир и стиль
- continuity-заметки
- уже использованные названия и хэштеги
- предложения новых названий и хэштегов
- редакторские замечания по развитию романа

Структура:
- `book/chapters/` : главы
- `book/context/` : контекст и память проекта
- `book/suggestions/` : варианты названий и хэштегов
- `book/editor_notes/` : редакторские замечания и векторы развития

Цель проекта:
использовать агентов для трех задач:
1. смысловой слепок новой главы
2. подбор названий и хэштегов
3. редакторская оценка главы как части романа в целом
