# Теги базы знаний для глав «Под Огромной Луной»

Каждая глава романа получает базовый тег:
- #под_Огромной_Луной

## Правила подбора тегов базы знаний

Для каждой главы романа, которая попадает в базу знаний, нужно ставить:

- обязательный тег `#под_Огромной_Луной`
- минимум 2 дополнительных содержательных тега базы знаний
- итого обычно не меньше 3 тегов

Дополнительные теги должны:
- соответствовать реальному содержанию главы
- быть полезными для поиска и навигации по базе
- отражать тему, тип динамики, проблему, внутренний конфликт или модель отношений
- не быть декоративными
- не браться механически из примеров

Нельзя:
- ограничиваться только тегом `#под_Огромной_Луной`, если по содержанию подходят другие теги
- автоматически повторять один и тот же дополнительный тег в разных главах
- использовать слабые или расплывчатые теги только потому, что они были в примере


Примеры:
- #семейные_ценности
- #на_тяге_чувств
- #избранные_сердцем

Эти теги не совпадают со сценическими тегами романа и выполняют другую функцию.
